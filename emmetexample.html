  1. Real Phyton:

    İçerisinde hem Python tutorial hem de ufak ufak bir sürü proje var. Yeni başlayanlar ve aliştirma yapmak isteyenler için harika

  2. Learn Python

    İçerisinde başliklara ayrilmiş şekilde Python tutoriallari var. Site içerisinde minik önerilerde de bulunmuşlar.

  3. Free Code Camp, Python Code Examples:

    Diğer önerilerden biraz farklı olsa da içerisinde örnekleri listelediği ve detaylandırdığı için güzel bir kaynak freecodecamp.org/news/python-code-examples-sample-script-coding-tutorial-for-beginners/

  4. Dev.to

    Günlük Python challange yapmak isterseniz harika bir kaynak bence. Gün gün ayrilmiş, derinlemesine konular ve örnekler var içerisinde

  5. Velit, architecto numquam.

  6. Yazbel İstihza

    Sanırım Python’a el süren herkesin bildiği bir kaynak Yazbel. İçerisindeki her konu tüm detaylarıyla tamamen Türkçe şekilde anlatılmış

  7. Programiz:

    İçerisinde vidolu, yazılı Python konu anlatımları ve örnekler bulunuyor. Yine her şeyin detaylıca işlendiği güzel bir kaynak
